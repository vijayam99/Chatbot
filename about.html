About This Project

This website hosts an AI-powered chatbot designed to provide compassionate emotional support and direct users to appropriate mental health resources.

It uses the Chatbase platform with OpenAI GPT for natural, empathetic conversation. A predefined system prompt ensures the chatbot remains supportive and sensitive.

Built using HTML, Tailwind CSS, and JavaScript for rapid development and easy maintenance. No user data is stored; all interactions are private.
